## 📲 Install on Your Phone (Step-by-Step)

### Method 1 — GitHub Pages (Recommended, Free)

1. **Create GitHub account** at github.com (free)
2. **Create new repository** → name it `bgmi-booster` → make it **Public**
3. **Upload all files** from this folder:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
4. Go to **Settings → Pages → Branch: main → Save**
5. Your app is live at: `https://YOUR-USERNAME.github.io/bgmi-booster`
6. **Open that link on your phone in Chrome**
7. Tap the **3-dot menu → Add to Home screen**
8. Done! App icon appears on your home screen ✅

---

### Method 2 — Convert to APK (using PWABuilder)

1. First complete Method 1 (get the GitHub Pages link)
2. Go to **pwabuilder.com** on PC
3. Paste your GitHub Pages URL → Click Start
4. Click **Android → Download Package**
5. Install the APK on your phone

## 📦 How to Find Package Name of Any App

1. Open **Google Play Store** on your phone
2. Search the game/app
3. Open its page — the URL will show the package name:
   `play.google.com/store/apps/details?id=**com.pubg.imobile**`
4. Copy that ID and paste in Manual Add
